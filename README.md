Vulnerability Assessment Report
Overview

This repository contains a vulnerability assessment report conducted on the web application testphp.vulnweb.com. The assessment was performed using Nmap, OWASP ZAP, and browser developer tools.

Objectives

Identify security vulnerabilities

Classify risks based on severity

Provide security recommendations

Tools Used

Nmap (Port scanning)

OWASP ZAP (Vulnerability scanning)

Browser Developer Tools (Header and cookie inspection)

Key Findings

Missing HTTP Security Headers (Medium Risk)

Insecure Cookie Configuration (Medium Risk)

Information Disclosure (Low Risk)

Unencrypted HTTP Communication (High Risk)

Potential Cross-Site Scripting (Medium Risk)
